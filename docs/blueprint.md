# **App Name**: Revelación Espiritual Interactiva

## Core Features:

- Quiz Dinámico: Un quiz gamificado con 15 preguntas para revelar la afinidad espiritual del usuario con un patriarca o profeta.
- Revelación Personalizada: Basado en las respuestas del usuario, el quiz revela un patriarca o profeta específico, junto con un mensaje inspirador adaptado a su perfil espiritual.
- Generación de Insights: Integra un modelo de lenguaje (tool) para ofrecer 'revelaciones' basadas en la comprensión del significado inherente a cada Patriarca o Profeta
- Página Post-Quiz con eBook: Después del quiz, los usuarios son dirigidos a una página que presenta un eBook relevante, diseñado para profundizar en los conocimientos adquiridos y ofrecer una experiencia espiritual más completa.
- Almacenamiento de Respuestas: Implementación de Firestore para almacenar las respuestas de los usuarios, permitiendo un análisis posterior de las tendencias y preferencias.
- Integración con Plataformas de Remarketing: Utilización de Google Tag Manager o Meta Pixel para rastrear el comportamiento de los usuarios y crear campañas de remarketing dirigidas.
- Diseño Responsivo y Optimización de Carga: Asegurar que la experiencia del usuario sea rápida y fluida, con un tiempo de carga máximo de 2 segundos en dispositivos móviles, y un diseño totalmente responsivo.

## Style Guidelines:

- Primario: Tono dorado suave (#CBB583) para evocar la antigüedad y lo sagrado.
- Fondo: Beige claro (#F5F5DC), sutilmente texturizado como pergamino antiguo, para dar un toque de misticismo y calidez.
- Acento: Ámbar (#FFBF00) para resaltar elementos clave y llamados a la acción, brindando luminosidad y enfoque.
- Títulos: 'Playfair' (serif) para un toque clásico y elegante. Note: currently only Google Fonts are supported.
- Cuerpo de texto: 'Lato' (sans-serif) para garantizar legibilidad y claridad en dispositivos móviles. Note: currently only Google Fonts are supported.
- Íconos lineales dorados, que representen elementos bíblicos y espirituales de manera minimalista y elegante.
- Diseño centrado y limpio, con cards animados para las preguntas y una barra de progreso intuitiva en la parte superior.
- Animaciones sutiles de 'revelación' después de cada respuesta, y transiciones suaves entre preguntas para mantener al usuario enganchado.
